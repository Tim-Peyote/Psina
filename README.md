# 🐾 Псина

Умный Telegram-бот с долговременной памятью, характером и умением жить в групповых чатах.

Не бот-помощник, а **живой участник чата** — слушает, запоминает, отвечает когда зовут, и молчит когда не просят.

---

## Что умеет

- **Помнит людей** — кто что любит, где работает, чем увлекается. Факты извлекаются автоматически из разговоров
- **Понимает контекст** — знает кто кому отвечает, о ком говорят, ведёт несколько разговоров параллельно не путая их
- **Живой характер** — верный, честный, с чувством юмора. Не подлизывается, может огрызнуться, не терпит издевательств
- **Подстраивается под чат** — формальный стиль или неформальный с матом, Псина адаптируется под вайб группы
- **Цензура настраивается** — пользователи сами решают уровень фильтров: строгий, умеренный или свободный
- **Напоминания** — «Псина, напомни завтра в 15 что совещание»
- **Дневные сводки** — краткий пересказ событий за день
- **Игровые сессии** — DnD-подобные игры с сохранением состояния
- **Конфиденциальность** — информация из одного чата никогда не попадёт в другой

---

## Как работает

```
Сообщение → Анализ контекста → Извлечение фактов → Детекция агрессии
→ Маршрутизация → Проверка уверенности → Анализ знаний → Генерация ответа
```

Псина **читает все сообщения**, но **отвечает только** когда:
- Позвали по имени (Псина, пёс, пёсик)
- Ответили на её сообщение
- Находятся в активной сессии диалога

Остальное — слушает и запоминает.

## Команды

| Команда | Описание |
|---------|----------|
| `/help` | Список команд |
| `/summary` | Дневная сводка |
| `/memory` | Что Псина помнит |
| `/profile` | Твой профиль |
| `/remind` | Создать напоминание |
| `/reminders` | Список напоминаний |
| `/mode` | Режим бота |
| `/censorship` | Уровень цензуры |
| `/vibe` | Вайб чата |
| `/game` | Игровые команды |
| `/silence` | Замолчать на N минут |

**Речью:** «заткнись», «будь поактивнее», «сбавь», «убери цензуру», «пофильтруй»

## Быстрый старт

```bash
# 1. Клон
git clone https://github.com/Tim-Peyote/Psina.git
cd Psina

# 2. Конфиг
cp .env.example .env
# Заполни TELEGRAM_BOT_TOKEN и LLM_API_KEY

# 3. Запуск
docker compose up -d
```

## Архитектура

```
src/
├── telegram_gateway/        # Приём сообщений из Telegram
├── message_processor/       # Нормализация сообщений
├── context_tracker/         # Отслеживание контекста разговоров
├── memory_engine/           # Память, факты, связи, профили
├── orchestration_engine/    # Мозг: маршрутизация, триггеры, сессии
│   ├── trigger_system.py    # Confidence scoring обращения
│   ├── session_manager.py   # Микросессии диалогов
│   ├── message_router.py    # Классификация сообщений
│   ├── knowledge_analyzer.py# Анализ знаний перед ответом
│   ├── vibe_adapter.py      # Подстройка под стиль чата
│   ├── censorship_manager.py# Управление цензурой
│   ├── abuse_detector.py    # Защита от издевательств
│   ├── personality.py       # Характер Псины
│   └── anti_chaos.py        # Защита от хаоса
├── retrieval_engine/        # Поиск контекста для LLM
├── llm_adapter/             # Qwen + Mock провайдеры
├── summarizer/              # Дневные сводки
├── game_engine/             # Игровые сессии
├── workers/                 # Celery: напоминания, сводки
├── admin_api/               # FastAPI админка
└── database/                # Модели, сессии, миграции
```

## Подробный запуск

### Локально

```bash
# Заполни .env (минимум: TELEGRAM_BOT_TOKEN)
cp .env.example .env

# Инфраструктура
docker compose up -d db redis

# Зависимости
pip install -e .

# Миграции
DATABASE_URL_SYNC=postgresql+psycopg2://postgres:postgres@localhost:5432/zalutka \
  alembic upgrade head

# Бот
python -m src.main
```

### Без LLM (mock режим)

```env
LLM_PROVIDER=mock
```

Бот будет работать, но отвечать заглушками. Для полноценной работы нужен Qwen или другой LLM.

### На сервере

```bash
# Docker
curl -fsSL https://get.docker.com | sh

# Клон
git clone https://github.com/Tim-Peyote/Psina.git && cd Psina

# Конфиг
cp .env.example .env
nano .env  # заполни токены

# Запуск
docker compose up -d
docker compose exec bot alembic upgrade head

# Логи
docker compose logs -f bot
```

### Обновление

```bash
git pull
docker compose up -d --build
docker compose exec bot alembic upgrade head
```

## Режимы работы

| Режим | Поведение |
|-------|-----------|
| `observer` | Молчит, пока не позовут |
| `assistant` | Отвечает на обращение или reply |
| `social` | Иногда вклинивается в разговор |
| `game_master` | Ведёт игровую сессию |
